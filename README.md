# Glamping Club

Лендинг глэмпинг-клуба
https://myapp-three-nu-33.vercel.app/

Проект демонстрирует навыки адаптивной верстки, построения компонентной архитектуры и разработки современного пользовательского интерфейса на React.

---

## Стек

- React
- TypeScript
- React Router
- SCSS (Sass)
- Vite
- Feature-Sliced Design (FSD)

---

## Возможности

- Адаптивная верстка
- Навигация между страницами
- Бургер-меню для мобильных устройств
- Современный интерфейс
- Переиспользуемые UI-компоненты
- Организация проекта по Feature-Sliced Design

---

## Архитектура

Проект построен по методологии **Feature-Sliced Design**.

```text
src/
├── app/
├── pages/
├── widgets/
├── features/
├── entities/
└── shared/
```

---

## Установка

Клонировать репозиторий

```bash
git clone <repository-url>
```

Перейти в папку проекта

```bash
cd glamping-club
```

Установить зависимости

```bash
npm install
```

Запустить проект

```bash
npm run dev
```
